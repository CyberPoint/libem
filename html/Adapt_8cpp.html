compute_expected_squares compute the squared-mean vectors weighted by the posteriors

Parameters:
X n by m array of data points
posteriors n by k matrix in which posterior densities will be placed, where k is the number of clusters
norm_constants the normalization constants (i-th constant is for i-th cluster)
[out] expected_squares vector of ptrs to mean-square matrices weighted by the posteriors (caller sets matrices to 0s)
Returns:
1 on success, 0 on error

note: the matrix we returned in the expected value (w.r.t norm constants) of a diagonal matrix whose i-th diagonal entry is given by the i-th component of the dot product of a data point with itself

00073 int adapt(int n, const double *X, vector<Matrix*> &sigma_matrix,

00079          * 1. construct an n X k "posterior" matrix of unnormalized weighted density values p_nk = P(n|k)*P(k)
00080          * for each data point, with each entry nk normalized by the total density for the n over all k.
00081         */

00089          *  data point, under the cluster.

00093          *  3. now compute the "alpha" constants a_i as v_i/ (v_i + relevance_factor).

00095

00097          *  4. now for each cluster, compute the cluster mean e_i as the weighted sum of the data point vectors,
00098          *  where the (n,k) posterior matrix entry is the weight for data point n and cluster k, and the expectation
00099          *  has normalizing constant v_i.
00100          */
00101

00103          *  5. now compute the "expected squares" matrix E_i for each cluster k, where by "expected square"
00104          *  we mean the expected value of the diagonal matrix whose (i,i)-th entry is given by the square of
00105          *  the i-th component of a randomly chosen data point n with itself, weighted by the (n,k) entry of the posterior
00106          *  matrix, and the expectation has normalizing constant v_i.
00107          *
00108          */
00109 
00110         /*
00111          *  6. now compute the new cluster weights W_i = Y [(a_i * v_i/N) + (1-a_i)*w_i

00113          */
00114

00116          *  7. now compute the new cluster means M_i as a_i * e_i + (1-a_i)*m_i where m_i is the old cluster mean
